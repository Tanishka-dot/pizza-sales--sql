Pizza Sales SQL Analysis

1- Project Overview-
This project analyzes a Pizza Sales database using SQL to uncover sales trends, customer preferences, and revenue insights by solving real-world business questions.

---

2- Project Objectives
- Analyze overall sales performance.
- Calculate total revenue and order trends.
- Identify the most popular pizzas and categories.
- Generate business insights using SQL queries.

---

3- Dataset
The project uses a Pizza Sales dataset consisting of four CSV files:

- `orders.csv`
- `order_details.csv`
- `pizzas.csv`
- `pizza_types.csv`

These datasets contain order information, pizza details, pricing, categories, quantities, and ingredients used for sales analysis.

---

4- SQL Analysis
- Total Orders
- Total Revenue
- Highest-Priced Pizza
- Most Ordered Pizza Size
- Top 5 Most Ordered Pizzas
- Category-wise Sales Analysis
- Hourly Order Distribution
- Average Pizzas Ordered Per Day
- Top Revenue-Generating Pizzas
- Revenue Contribution Analysis
- Cumulative Revenue Analysis

---

5- Key Insights
- Identified the highest revenue-generating pizzas.
- Analyzed customer ordering patterns.
- Compared sales across pizza categories.
- Determined peak ordering hours.
- Evaluated revenue contribution by pizza type.
